# UndergroundGarage_RUS
Руссификатор для игры Underground Garage

Поместить файл rusLock_P.pak по пути "Папка с игрой/UndergroundGarage/Content/Paks/"

Итоговый путь должен получится таким: "E:\Games\Underground Garage\UndergroundGarage\Content\Paks\rusLock_P.pak"

Перевод выполнен "кривым" переводчиком "Яндекс". Могут присутствовать неточности.
